# HOW TO USE
## Inputting the words to use
How this works is it scrambles up all the words into every combination possible. All you need to do is input the words for it to scramble. When you input the words seperate them with a comma and a space.
**Example:**
apple, banana, pear

There are also presets you can use, to get them click **Presets** at the bottom and click the one you want.
**Warning: Do not input more than 5-6 words or else the program might freeze up.**
## Output
When done, click **Generate** to output the words. The words will appear below for you to copy.
## Options:
### Include hashtags?
This put hastags before the generated results.
**Hashtags ON:**
#python #programming #hashtags
**Hashtags OFF:**
python programming hashtags
Hashtags off can be used to put them at the bottom of YouTube descriptions.
Hashtags on can be used for TikTok.

### Seperate by Comma?
This puts commas before the generated results.

**Seperate with commas ON:**
python, programming, hashtags
**Seperate with commas OFF:**
python, programming, hashtags
Seperate with commas on can be used for YouTube tags.
Seperate with commas off can be used for pasting a bunch of tags into the title (e.g. TikTok.)

**You can also combine them!**
#python, #programming, #hashtags

*There are also examples in the program.*
